Продукт

Разрабатываем Plesk: панель управления серверами и веб-приложениями, продаваемую в 140 странах мира.

Не слышали о нас? Нужно срочно это исправить, так как 6% веб-сайтов в Интернете хостятся на Plesk.

Наш R&D работает в Новосибирске, Академгородке и Кёльне. Пишем на PHP, JS, C++, Go, Python и Java.

Сотрудничаем с Amazon Web Services, Google Cloud, Alibaba Cloud, Microsoft Azure и другими компаниями, входящими в Top 100 хостинговой индустрии.

Команда

Наша команда занимается созданием ценности продукта, продумывает, как сделать так, чтобы пользователям было удобно, чтобы пользователей было больше. Доводилось видеть продукты, сделанные неудобно? Так вот именно Plesk Program Manager Team отвечает за то, чтобы продукт решал задачи пользователей наилучшим образом. Изучает потребности, формулирует решения, проверяет результат.

Здесь слушают и спрашивают клиентов, генерируют и проверяют гипотезы, делают выводы из метрик, ищут точки роста и планируют роадмапы.

Здесь важно быть самостоятельным, иметь системное мышление, умение критически смотреть на идеи и решения, откидывать то что не работает и оставлять только то что работает.

Почему это может быть вам интересным?

  • Вы перед собой видите challenge работать с зарубежными рынками Европы и США
  • Plesk продукт с аудиторией в 350K серверов и 11млн+ сайтов по всему миру (да по меркам мобильных приложений это не много, но для web hosting это 6% всех сайтов мирового интернета)
  • Если у Вас есть амбиции строить решение по анализу данных и оказывать влияние на пользователей по всему миру (Евросоюз, США, Азия) то Вам представится такая возможность.

Какие задачи предстоит решать?

  • Построение рекомендательной системы для пользователей продукта.
  • Формирование персонализированных рекомендаций в продукте на основе пользовательских действий.
  • Формирование персонализированных предложений в Marketplace на основле использования продукта.
  • Построение системы анализа и предсказания критических действий пользователя (churn, обращение в саппорт).
